# NorthernRockiesChitoRyu
- https://pg-chito-ryu-azcbn.ondigitalocean.app/index.html

This project was created to develop hands-on experience with web development, hosting, and version control while supporting a local karate dojo.

## Overview
- Designed and deployed a public-facing website
- Hosted on DigitalOcean
- Initially developed using ASP.NET Core, later redesigned using HTML5 for cost efficiency

## Key Features
- Static website structure using HTML5, CSS, and JavaScript
- Organized content for dojo information and communication
- Lightweight design optimized for low-cost hosting

## Technical Decisions
- Transitioned from ASP.NET Core to static HTML to eliminate hosting costs
- Evaluated trade-offs between dynamic functionality and operational cost

## Tools & Technologies
- HTML5, CSS, JavaScript
- GitHub (version control)
- DigitalOcean (hosting)

## Future Improvements
- Reintroduce dynamic features (forms, scheduling, etc.)
- Enhance UI/UX
- Explore backend integration options
- Expand UI and interactivity
